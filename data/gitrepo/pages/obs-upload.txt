====== Open Bible Stories Upload ======

You can upload a single DOCX file or a single zipped DOCX file (.docx or .zip). Currently only Word documents in the DOCX format can be processed.

The size of the file being uploaded must be less than 1 gigabyte (1024 megabytes).

OBS must already be initialized for the language you select.  [[obs-setup|Check here]].

If you need help getting started, please [[:contact|contact us]].

If the target language does not appear in the search, go to [[http://td.unfoldingword.org/uw/languages|unfoldingWord translationDatabase]]. Type in the name of the target language in the search field on that page. Enter the code, from the far left column, in the Language field below.
